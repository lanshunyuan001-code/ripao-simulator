# 日抛模拟器

纯前端单页网页小游戏，使用 HTML / CSS / JavaScript 实现，无需后端或构建步骤。

## 当前状态

游戏内容已清空为新的约会健康风险模拟框架。

已写入的基础内容：

- 数值：生理欲望、心理负荷。
- 行动：喝杯酒 / 套套话、戴套口交、戴套性交、无套口交、无套性交。
- 合规声明保留。

等待补充的内容：

- 约会对象提示词。
- 疾病与风险规则。
- 行动后的反馈文案。
- 胜利、失败与复盘文案。
- “喝杯酒 / 套套话”何时可用的规则。

## 本地运行

```bash
py -m http.server 5173
```

然后访问 `http://localhost:5173`。
